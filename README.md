# Payroll Management System

## Description
The Payroll Management System is a desktop application developed using Python and Tkinter. It helps in managing employee salary details by automatically calculating gross pay, tax, pension, and net salary. The system also allows storing employee data for future reference.

---

## Objective
To design a simple and efficient system that automates salary calculations and reduces manual errors in payroll management.

---

## Features
- Enter employee details (Name, Address, Post Code, Gender)
- Input salary components (Basic, Overtime, Other payments)
- Automatic calculation of:
  - Gross Salary
  - Tax (10%)
  - Pension (5%)
  - Net Salary
- Save employee data into a text file
- Reset option to clear inputs
- Exit button to close application

---

## Technologies Used
- Python
- Tkinter (GUI)
